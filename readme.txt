Abilita Postapay (o altro sistema di ricarica) come sistema di pagamento per WooCommerce. 

== Description ==

Usate WooCommerce per il vostro ecommerce? 
Aggiungete Postepay tra le opzioni di pagamento. 
Scegliendo questa opzione in fase di check-out verrà mostrato il vostro numero di carta Postepay e l'acquirente potrà fare la ricarica.

Può essere utilizzato anche per altri sistemi di ricarica, basta modificare l'intestazione e la descrizione.

*Richiede WooCommerce per funzionare*

*Richiede WooCommerce 2.0.20 - Testato fino a WooCommerce 3.4.3*


== Installation ==


1. Carica 'woocommerce-postepay' nella cartella '/wp-content/plugins/' 
2. Attiva il plugin dal menu 'Plugins' di WordPress
